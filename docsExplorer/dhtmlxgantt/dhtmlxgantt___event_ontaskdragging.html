Event Handling

onTaskDragging

onTaskDragging event is fired while user is dragging a Task item.

One parameter is passed to the handler:

  • task - GanttTask object which is being dragged.

You may return false from this handler to stop dragging. 

    var dayX = new Date(2010, 5, 18);
    ganttChartControl.attachEvent("onTaskDragging", function(task,newdate){
        if (newdate > dayX) {
            // Do not drag outside of date 5/18/2010
            return false;
        }
    });
